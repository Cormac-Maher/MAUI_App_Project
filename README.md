Movie Explorer — .NET MAUI Cross‑Platform App

Description:
Movie Explorer is a cross‑platform mobile and desktop application built with .NET MAUI.
It allows users to browse, search, sort, and manage a personalised movie collection.
This app was developed as part of a 2nd Year Cross‑Platform Application Project, focusing on UI design, data persistence, and multi‑platform compatibility.

Features:
Browse a list of movies
Search movies by title
Sort movies by rating, year, director, genre, or title
Add custom movies
Remove movies
Favourite movies
View history 
Emoji based movie quiz mode
Change background colour
JSON data downloaded and cached locally
Separate favourites/history per user
Runs on Windows and Android

Technologies used:
Programming Languages - C# & XAML
Framework - .NET MAUI
Data storage using JSON
Made on Visaul Studio 2022

How to set up the App:
You must have Visual Studio 2022 with .NET Multi‑platform App UI workload
Clone this repository in Visual Studio Code
Run the app on Windows Machine or Android emulator

How to use the app:
1. Enter Your Name
When the app launches, enter your name.
This ensures favourites and history are stored separately for each user.

2. Main Page — Browse Movies
All movies are displayed with basic info
Search bar to search by title
Sorting options for rating, year, director, genre, and title

3. Settings Page
Change background colour

4. Quiz Mode
Match the emoji to the correct movie!
A fun interactive feature using your moviesemoji.json dataset.

5. Favourites Page
Shows all movies the user has favourited
Stored in a local JSON file

6. History Page
Automatically logs every viewed movie
Includes timestamps
Stored per user

7. Add Custom Movie
Users can add custom movies
Supports title, year, rating, genre, and image

8. Remove Movies
Movies can be deleted from the collection
Updates the JSON file accordingly
